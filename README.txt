This program assists construction companies in keeping track of various job's expenses including employee payroll.
Users create projects corresponding to real life construction projects.
Users then enters expenses per each project on a weekly basis.
The program keeps track of all the data, and generates graphs displaying profit and loss.
Program also displays reports of employee payments. Users can add or remove employees. Create and delete projects.

I wrote this program back in 2017, shortly after I learned JavaFX.
I'm sure there are a lot of noob mistakes in here, I am still learning.

I created my own product key verification system, not sophisticated at all.
You can find keys in the src/ProgramFiles/Key.txt file.
